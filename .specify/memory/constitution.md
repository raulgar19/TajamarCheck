# CheckingTajamar Constitution

## Core Principles

### I. Arquitectura Desacoplada
La solución debe mantener el backend como una API REST limpia y estructurada. El frontend Angular se conecta a la API utilizando únicamente peticiones HTTP a través de servicios dedicados.

### II. Lenguaje e Idioma
El código fuente (nombres de variables, clases, identificadores técnicos) debe estar en inglés. Los mensajes mostrados en la interfaz gráfica del usuario (Angular) y las respuestas explicativas del servidor deben estar en español. El gestor de paquetes oficial del proyecto es `pnpm`.

### III. Seguridad y Autenticación
El sistema expone un mecanismo seguro de autenticación (Login) a través del backend. Se prohíben atajos de seguridad en la presentación y el estado de la sesión debe controlarse adecuadamente tanto en la API como en la SPA.
